# SlidingPuzzle
A Sliding Tile Puzzle to run on an Android device.

## Installation
### Prerequisites
Install Git and Android Studio on local machine.

### Setup
- Clone the repository in Android Studio.
- Generate `.apk` file.
- Download on android mobile devices.
